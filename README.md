Shark Tank Analytics

Project Description

Shark Tank Analytics is a data analysis project that examines startup investment trends using a Shark Tank dataset. The project leverages Python-based data analysis and visualization techniques to uncover insights related to startup valuations, funding amounts, investors, industries, and deal outcomes.

Objectives

- Analyze startup investment patterns.
- Identify industry-wise funding trends.
- Understand investor participation and deal success rates.
- Explore the relationship between startup valuation and investment received.

Tools & Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

Dataset Information

The dataset contains information about:

- Startup Industry
- Ask Amount (Lakhs)
- Investment Received (Lakhs)
- Startup Valuation (Lakhs)
- Investor
- Deal Status

Analysis & Visualizations

- Industry-wise Analysis
- Deal Status Analysis
- Investor Distribution
- Investment Amount Distribution
- Startup Valuation Analysis
- Correlation Analysis

Visualizations Used

- Bar Charts
- Pie Charts
- Histograms
- Box Plots
- Scatter Plots
- Heatmaps
- Pair Plots

Key Findings

- Certain industries attracted more investments than others.
- Startup valuation showed a positive relationship with investment received.
- Investor participation varied significantly across startups.
- Deal status influenced funding outcomes.

Conclusion

This project demonstrates the application of Exploratory Data Analysis (EDA) techniques to derive meaningful insights from startup investment data and supports data-driven decision-making.

Project Type: Data Analytics Project
Tools: Python | Pandas | Matplotlib | Seaborn
